# PA Agent — 价格行为 AI 分析桌面终端

面向主观交易者的 **价格行为（Price Action）** AI 辅助决策桌面工具。直接从 **MT5 / TradingView / 东方财富** 读取 K 线数据，将结构化 K 线特征送入大模型做**两阶段分析**（市场结构诊断 → 交易决策），**不是**截图识图，**不连接券商、不执行下单**。

---

## 核心功能

### 两阶段 AI 分析
- **阶段一（市场诊断）**：识别周期位置（spike / channel / trading range 等 8 类）、闸门规则判定、逐 K 线标注、多空方向判断
- **阶段二（交易决策）**：策略文件路由 → 经验库检索 → 生成交易方案（入场/止损/止盈、限价单/突破单/市价单/不下单）+ 未来走势预期
- **增量分析**：新 K 线收盘后基于上轮结论增量更新，无需全量重跑
- **持续跟踪**：开启后新 K 线收盘自动触发新一轮分析

### 多数据源
| 数据源 | 适用市场 | 平台 |
|--------|----------|------|
| MT5 | 外汇、黄金、指数、期货 | Windows |
| TradingView | 外汇、商品、美股、A股、港股 | 全平台 |
| 东方财富 | A 股（日/周/月 + 分钟线） | 全平台 |

### A 股板块批量扫描
- 选择板块类型（行业板块/概念板块/地域板块）
- 按市值/涨幅/跌幅/量比/市盈率排序，取前 N 只
- 逐只自动进行两阶段 AI 分析
- 统计汇总：方向分布、周期分布、下单类型分布、平均置信度/胜率
- **未来走势预期统计**：下一根 K 线方向 + 平均概率；下一市场周期方向 + 周期分布
- 点击股票名称查看 K 线蜡烛图

### 决策树与可视化
- 内置二元决策树（37 个闸门节点），AI 每一步的判定路径完整可追溯
- 赛博科幻风格可视化流程图，支持自动播放动画
- 调试面板显示每次分析的完整 Prompt、原始响应、诊断 JSON

### 分析后自由追问
- 分析完成后可针对结果进行不限轮次的自由对话
- AI 可结合 K 线快照回答追问
- 对话历史持久化保存

### 经验库系统
- 按周期位置（spike / channel / trading range / TR 等）检索历史案例
- AI 分析时自动加载匹配的经验条目作为参考
- 成功/失败案例分别归档

### 完整落盘与校验
- Prompt、原始响应、诊断/决策 JSON、Token 用量、追问记录全部持久化
- JSON Schema 校验 + 跨字段一致性检查 + 语义校验
- 校验失败自动重试（可配置重试次数与策略）

## 快速开始

```bash
# 安装依赖
pip install -e .

# 启动
python run.py
```

首次启动后在 **菜单 → AI 模型设置** 中填写 **Base URL**、**模型名** 与 **API Key**。

> 如需隔离环境：`python -m venv .venv` → 激活 → `pip install -e .`

### 数据源配置

- **MT5**：需安装 MetaTrader 5 终端并登录账号
- **TradingView**：无需额外配置，选择交易所后可直接获取行情
- **东方财富（A 股）**：内置支持，直接输入 6 位股票代码即可

## 项目结构

```
pa_agent/
├── ai/                 # AI 分析引擎（Prompt 组装、决策树、校验）
├── config/             # 配置模块
├── data/               # 数据源（MT5 / TV / 东方财富）
├── gui/                # PyQt6 桌面界面
│   ├── widgets/        # K 线蜡烛图、状态条等 UI 组件
│   ├── batch_scan_dialog.py    # A 股板块批量扫描
│   ├── kline_view_dialog.py    # K 线图查看器
│   └── main_window.py          # 主窗口
├── orchestrator/       # 两阶段编排器
├── records/            # 分析记录持久化
├── prompt_engineering/ # 策略文件（22 个交易策略）
└── experience/         # 经验库案例
```
